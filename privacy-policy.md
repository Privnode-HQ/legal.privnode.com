# 隐私政策

**生效日期：2025年12月15日**

**Agent Velo, LLC**（以下简称“**本公司**”、“**我们**”、“**Privnode**”或“**供应方**”）尊重您的隐私，并致力于通过遵守本隐私政策来保护您的个人数据。

本隐私政策旨在明确说明我们如何收集、使用、处理、披露及保护您在访问 https://privnode.com/（“**网站**”）或使用 Privnode API 聚合平台及相关服务（“**服务**”）时提供的个人信息。本政策中使用的未定义术语，其含义与我们的《服务条款》中的定义相同。

在使用本服务前，请您仔细阅读本政策。**通过访问或使用本服务，即表示您同意本政策的全部条款。**


## **1. 我们收集的信息**

我们收集信息的目的是为了向您提供服务、改进用户体验以及保障系统安全。

**1.1. 您主动提供的信息**
当您注册账户、配置服务或联系我们时，您会直接向我们提供特定信息。这主要包括您的账户注册信息，例如您的电子邮件地址、用户名、密码，以及您可能选择提供的姓名或公司名称。当您购买积分（Credits）时，我们或我们的第三方支付处理商（如 Stripe 或 Coinbase）会收集必要的计费信息，包括支付详情、账单地址和交易记录。此外，如果您通过电子邮件或客户支持渠道与我们联系，我们将收集您通信的内容以及您提供的任何联系方式。**特别需要注意的是**，当您使用 API 发送请求时，您提交的文本提示（Prompt）、文件、代码片段（统称“**输入**”）以及服务生成的响应（“**输出**”）均由我们的系统处理；如果这些内容中包含个人数据，该数据将被视为我们收集的信息的一部分。

**1.2. 自动收集的信息**
当您访问网站或通过 API 接口调用服务时，我们利用自动数据收集技术获取特定的技术数据。这些数据包括您的设备与网络信息，如 IP 地址、浏览器类型、操作系统版本、设备标识符及时区设置。同时，我们会收集使用数据（Log Data），这涵盖了 API 请求的时间戳、请求路径、错误代码、响应时间、消耗的 Token 数量以及点击流数据。

**1.3. Cookie 与追踪技术**
我们使用 Cookie、像素标签和类似技术来识别您的身份、保存偏好设置及分析流量。我们将这些技术分为三类：首先是绝对必要的 Cookie，它们对于维持登录状态、确保支付安全及核心功能运行至关重要，无法在系统中关闭；其次是功能性 Cookie，用于记住您的语言偏好、API 配置等，以提供个性化体验；最后是分析性 Cookie，用于收集匿名化的统计数据（例如通过 Google Analytics 或 PostHog），帮助我们了解用户如何使用服务并据此进行优化。


## **2. 我们如何使用您的信息**

我们仅在有合法依据的情况下处理您的个人信息，具体目的如下：

**2.1. 提供核心服务与账户管理**
我们处理您的信息主要是为了执行与您签订的合同，包括处理 API 请求、将您的输入路由至第三方模型提供商、计算 Token 消耗以及执行计费扣款。同时，我们使用您的联系信息来验证您的身份、发送重要的账户通知（如余额不足提醒、安全警报）以及处理密码重置请求。

**2.2. 服务改进、安全与合规**
为了维护服务的稳定性与安全性，我们分析聚合的、去标识化的使用趋势，以优化 API 性能和路由算法。我们监控网络流量以防止 DDoS 攻击、爬虫滥用或欺诈行为，并检测违反《服务条款》的行为（如生成非法或 NSFW 内容）。此外，我们可能需要处理您的信息以履行法律义务，例如税务申报、审计要求或响应执法机关的合法请求。

**2.3. 关于模型训练的声明**
**Privnode 默认不会使用您的输入（Inputs）或输出（Outputs）来训练我们自己的大语言模型。** 除非您主动开启了平台提供的“日志记录”或类似的选择性加入（Opt-in）功能，否则我们不会为了模型训练目的而存储您的内容数据。


## **3. 信息的共享与披露**

我们承诺不会出售您的个人数据。我们仅在以下必要情形下共享您的信息：

**3.1. 第三方模型提供商**
由于 Privnode 是一个聚合平台，为了生成您请求的内容，我们必须将您的输入（Inputs）传输给相应的第三方模型提供商（例如 OpenAI, Anthropic, Google, Meta 等）。一旦数据传输至第三方，该数据的处理将受该第三方的隐私政策约束。我们强烈建议您查阅相关模型提供商的隐私条款，以了解其关于数据保留和训练的具体政策。

**3.2. 服务提供商**
我们聘请受信任的第三方公司协助运营服务，包括用于存储数据和运行基础设施的云托管服务商、处理信用卡或加密货币交易的支付处理商，以及帮助我们分析网站流量的分析工具供应商。这些服务提供商仅有权在代表我们执行特定任务时访问您的信息，并受严格的保密义务约束，不得将数据用于任何其他目的。

**3.3. 法律要求、安全与业务转让**
如果我们要善意地认为披露是必要的，以遵守法律程序或政府要求、执行我们的服务条款、或保护 Agent Velo, LLC 及其用户或公众的权利、财产或安全，我们可能会披露您的信息。此外，如果涉及合并、收购、资产出售或重组，您的用户信息可能作为转让资产的一部分被转移给继受方。


## **4. 您的权利与选择**

根据您所在的司法管辖区（如 GDPR 或美国各州法律），您对自己的个人信息享有多项权利。

**4.1. 访问、更正与删除**
您有权请求获取我们持有的关于您的个人数据副本，要求更正不准确或不完整的个人信息，或请求删除您的账户及相关数据（即“被遗忘权”），但在法律要求我们必须保留数据（如税务记录）的情况下除外。

**4.2. 限制处理与数据可携带性**
在特定情况下，您有权要求限制我们处理您的数据，或索取结构化、机器可读格式的数据副本以便传输给其他服务商。

**4.3. 撤回同意与行使权利**
对于基于同意处理的数据（如接收营销邮件），您可随时撤回同意。若您希望行使上述任何权利，可以通过发送电子邮件至 **privacy@privnode.com** 提交请求。为保障账户安全，我们可能会在处理请求前要求验证您的身份。


## **5. 数据保留**

我们仅在实现收集目的所需的时间内保留您的个人信息，除非法律要求延长保留期。对于输入与输出数据，Privnode 的默认策略是仅在传输和处理请求所需的短暂时间内在内存中处理，不会将其持久化存储到数据库中，除非用于必要的元数据记录（如 Token 计数）。唯一的例外是如果您主动开启了“历史记录”或“调试模式”，我们将按照您的配置在指定期限内存储您的详细日志。当您请求删除账户后，我们将在 30 天内删除或匿名化您的个人信息，但备份档案中的残留数据可能需要更长时间才能完全清除。


## **6. 数据安全**

我们采取符合行业标准的技术措施，包括传输层安全协议（TLS/SSL）、静态数据加密和严格的访问控制，以保护您的数据免受未经授权的访问、丢失或滥用。然而，由于互联网传输和电子存储存在固有风险，我们无法保证数据绝对安全。您有责任保护您的账户凭证（尤其是 API Key）的安全。若发现任何潜在的安全漏洞，请立即通知我们。


## **7. 国际数据传输**

**Agent Velo, LLC** 位于美国。无论您身在何处，您的信息都可能被传输至美国或我们的服务提供商所在的国家/地区进行处理和存储。如果您位于欧洲经济区（EEA）、英国或瑞士，我们将依据欧盟委员会批准的《标准合同条款》（SCCs）或适当的法律机制来保障跨境数据传输的安全性，确保您的数据获得与在您本国同等的保护水平。


## **8. 儿童隐私**

我们的服务严格限制 **18 周岁及以上** 的用户使用。我们不会故意收集 13 岁以下儿童的个人信息。如果我们发现无意中收集了未成年人的信息，我们将立即采取措施予以删除。如果您认为我们可能持有任何未成年人的信息，请通过下方的联系方式通知我们。


## **9. 第三方链接与政策更新**

本服务可能包含指向第三方网站或服务的链接，我们不对这些第三方的隐私惯例负责。我们可能会不时更新本隐私政策以反映业务变化或法律要求。任何重大变更将通过电子邮件通知您或在网站显著位置发布公告。修订后的政策自页面顶部显示的“生效日期”起生效，继续使用服务即视为您接受更新后的政策。
